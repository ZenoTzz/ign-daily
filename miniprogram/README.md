# IGN Daily 微信小程序

这是 IGN Daily 的微信小程序客户端，直接连接服务器 API：

```text
https://igndaily.site/api
```

## 你需要做的事

1. 安装并打开微信开发者工具。
2. 选择“导入项目”，项目目录选这个仓库里的 `miniprogram` 文件夹。
3. 如果只是本地预览，可以先用测试号；如果要真机/发布，使用你自己的微信小程序 AppID。
4. 在小程序后台把 `https://igndaily.site` 加到 request 合法域名。
5. 用网站里的服务器账号登录小程序。

## 当前功能

- 微信管理员静默登录；首次使用通过现有服务器管理员账号完成绑定
- 按新闻日查看文章队列，支持状态筛选和批量选择
- 提交 Codex 翻译并查看实时任务进度
- 提交时可授权一次“翻译完成”订阅提醒（不发送新闻提醒）
- 独立任务中心，可恢复最近任务
- 查看复核状态、译文摘要、按需展开配图和完整正文，支持上一篇/下一篇
- 搜索正式词库并提交待审核译名候选
- 查看服务器及微信绑定状态、退出登录；账号修改保留在主站

## 页面结构

```text
工作台  今日文章、状态筛选、批量翻译
任务    最近任务、逐篇进度、失败原因和文章直达
词库    搜索正式词库、提交候选
账号    登录与绑定状态、退出登录
详情    复核信息、摘要、折叠配图、正文和前后篇导航
```

## 发布前检查

1. 微信公众平台 → 开发管理 → 服务器域名，将 `https://igndaily.site` 加入 `request` 合法域名。
2. 微信开发者工具中确认 `project.config.json` 的 AppID 属于当前小程序。
3. 在小程序后台选择“翻译完成”订阅消息模板，将模板 ID 配置为服务器的 `IGN_DAILY_WECHAT_JOB_TEMPLATE_ID`。
4. 真机测试登录、文章读取、提交翻译、订阅授权和候选词提交。
5. 开发者工具点击“上传”，再到公众平台提交体验版或审核。

小程序不会保存 GitHub PAT、API Key 或 SSH 密钥，只保存服务器返回的登录 Token。
微信 AppSecret 只保存在服务器 `/srv/ign-daily-api/.env`，绝不能写入小程序代码。
