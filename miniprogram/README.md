# IGN Daily 微信小程序

这是 IGN Daily 的微信小程序客户端，直接连接服务器 API：

```text
https://igndaily.site/api
```

## 你需要做的事

1. 安装并打开微信开发者工具。
2. 选择“导入项目”，项目目录选这个仓库里的 `miniprogram` 文件夹。
3. 如果只是本地预览，可以先用测试号；如果要真机/发布，使用你自己的微信小程序 AppID。
4. 在小程序后台把 `https://igndaily.site` 加到 request 合法域名。
5. 用网站里的服务器账号登录小程序。

## 目前已支持

- 服务器账号登录
- 按新闻日查看文章列表
- 勾选文章并提交翻译
- 查看最近翻译任务进度
- 查看文章详情
